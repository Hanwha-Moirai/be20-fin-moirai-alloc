# Alloc  
인력을 효율적으로 배치하는 프로젝트 운영 시스템

---

## 📌 프로젝트 개요
Alloc은 **프로젝트 인력 배치**와 **일정/진척 관리**를 중심으로, PM이 안정적으로 프로젝트를 운영할 수 있도록 돕는 시스템이다.

## 🧩 핵심 기능
- 프로젝트 생성 및 기본 정보 관리
- 인력 배치(역할/투입률/기간 기반)
- 일정 관리(캘린더/간트/마일스톤 등)
- 진척 현황 가시화 및 변경 이력 관리(필요 시)

- Frontend: Vue
- Database: MariaDB, Qdrant
- AI: Langchain, QWEN
- Infra: AWS, Docker
